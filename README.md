# THIS IS A PIP TRAINING 

# PLAYWRIGHT 

## Requirements

- NodeJS
- VSCode
- Playwright Extension
- Code Runner Extension (Optional)

## Steps 

1. Install NodeJS 
2. Install VSCode
3. Install Playwright Extension
4. Clone this repo
5. Write on the terminal: npm init playwright@latest
6. To run test you can use either the terminal writing npx playwright test (it will run all tests in test folder), or npx playwright test filename.spec.ts for single test run. Also you can use the extension to run and debug
7. In practice folder there are examples, to run them you need to drag them to tests folder
